# Funções


**Google Web Fonts**

`$font-url : "http://fonts.googleapis.com/css?family=Rokkitt:400,700";`

---


**Converter valores REM**  

`propriedade: rem(16px);`

---


**Converter valores EM**  

`propriedade: em(16px);`

---


**Separa valores**

`strip-units(16px); (Resultado: 16)`
